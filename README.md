# 🗳️ ProjectVote

**ProjectVote** to prosta aplikacja webowa umożliwiająca użytkownikom **głosowanie na projekty** oraz przeglądanie ich aktualnego rankingu.
Projekt został stworzony jako aplikacja edukacyjna do nauki pracy z **bazą danych, backendem i frontendem**.

---

## ✨ Funkcje

* 📋 Wyświetlanie listy dostępnych projektów
* 👍 Oddawanie głosów na wybrane projekty
* 📊 Automatyczne zliczanie głosów
* 🏆 Ranking projektów według liczby głosów
* 🔄 Aktualizacja wyników

---

## 🖥️ Demo

Przykładowy wygląd aplikacji:

```
+--------------------------------------+
|        Lista projektów               |
+--------------------------------------+
| 1. System zarządzania biblioteką     |
|    👍 12 głosów                      |
|                                      |
| 2. Aplikacja do nauki języków        |
|    👍 8 głosów                       |
|                                      |
| 3. Gra edukacyjna                    |
|    👍 5 głosów                       |
+--------------------------------------+
```

Użytkownik wybiera projekt i klika **Głosuj**, a system zapisuje głos w bazie danych.

## 🛠️ Technologie

Projekt może korzystać z następujących technologii:

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python
